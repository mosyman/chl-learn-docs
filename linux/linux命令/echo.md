### 一、echo 命令核心定位
`echo` 是 Linux 中最基础的命令之一，核心作用是<span style="color: #ff6600; font-size: 15px; font-weight: bold;">在终端输出指定的字符串、变量值或特殊字符</span>，常用来调试脚本、输出提示信息、写入文件等。
- 语法格式：`echo [选项] [输出内容]`
- 核心特点：简单、轻量，是 Shell 脚本中输出信息的“标配”。

### 二、基础用法（无选项）
#### 1. 输出普通字符串
直接跟字符串（可加引号，也可不加，建议加引号避免特殊字符解析问题）：
```bash
# 无引号（简单字符串可用）
echo hello world
# 输出：hello world

# 单引号（原样输出，不解析变量/特殊字符）
echo 'hello $USER'
# 输出：hello $USER

# 双引号（解析变量/转义字符，推荐）
echo "hello $USER"
# 输出：hello root（假设当前用户是 root）
```

#### 2. 输出变量值
`echo` 支持解析 Shell 环境变量、自定义变量，用 `$变量名` 引用：
```bash
# 输出系统环境变量
echo "当前用户：$USER，当前目录：$PWD"
# 输出：当前用户：root，当前目录：/root

# 输出自定义变量
name="张三"
echo "姓名：$name"
# 输出：姓名：张三
```

### 三、核心选项（常用）
`echo` 的选项不多，最常用的是 `-n` 和 `-e`，其他选项几乎不用，重点掌握这两个：

| 选项 | 作用 | 示例 | 输出 |
|------|------|------|------|
| `-n` | 不输出末尾的换行符（默认 echo 会换行） | `echo -n "hello"` | `hello`（光标停在该行末尾） |
| `-e` | 启用转义字符解析（如 \n 换行、\t 制表符） | `echo -e "hello\nworld"` | hello<br>world |

#### 选项实战示例
```bash
# -n 示例：连续输出不换行
echo -n "请输入密码："
read password  # 读取用户输入，光标紧跟在"请输入密码："后
# 输出：请输入密码：_（光标在下划线位置）

# -e 示例：解析转义字符
echo -e "姓名\t年龄\t性别\n张三\t20\t男\n李四\t22\t女"
# 输出：
# 姓名    年龄    性别
# 张三    20      男
# 李四    22      女

# -e 解析特殊转义符
echo -e "退格：hello\bworld"  # \b 退格（删除前一个字符）
# 输出：hellworld（删除了 hello 后的 o）
echo -e "换页：hello\fworld"  # \f 换页
echo -e "回车：hello\rworld"  # \r 回车（覆盖当前行开头）
# 输出：world（覆盖了 hello）
```

### 四、高级用法（实战场景）
#### 1. 输出内容到文件（重定向）
结合 `>`（覆盖）、`>>`（追加）重定向，将 echo 输出写入文件：
```bash
# 覆盖写入文件（文件不存在则创建，存在则清空原有内容）
echo "hello world" > test.txt

# 追加写入文件（保留原有内容，新增内容到末尾）
echo "新增内容" >> test.txt

# 查看文件内容验证
cat test.txt
# 输出：
# hello world
# 新增内容
```

#### 2. 输出命令执行结果（命令替换）
用 `` `命令` `` 或 `$(命令)` 嵌套执行命令，echo 输出结果：
```bash
# 输出当前时间
echo "当前时间：$(date +%Y-%m-%d\ %H:%M:%S)"
# 输出：当前时间：2026-02-22 19:30:00

# 输出当前目录下的文件列表
echo "当前目录文件：$(ls)"
# 输出：当前目录文件：test.txt main.sh
```

#### 3. 结合管道（|）传递内容
将 echo 输出作为其他命令的输入：
```bash
# 输出字符串并转为大写
echo "hello world" | tr 'a-z' 'A-Z'
# 输出：HELLO WORLD

# 输出数字并计算求和
echo "1 2 3 4 5" | awk '{sum=0; for(i=1;i<=NF;i++) sum+=$i; print sum}'
# 输出：15
```

#### 4. 输出彩色文字（终端美化）
结合 ANSI 转义码，用 `-e` 输出彩色文字（脚本提示常用）：
```bash
# 格式：echo -e "\033[字体样式;字体颜色;背景颜色m 内容 \033[0m"
# 常用颜色：31(红)、32(绿)、33(黄)、34(蓝)、35(紫)、36(青)、37(白)

# 绿色文字（成功提示）
echo -e "\033[32m执行成功！\033[0m"

# 红色背景+白色文字（错误提示）
echo -e "\033[41;37m执行失败！\033[0m"

# 加粗+黄色文字（警告提示）
echo -e "\033[1;33m警告：内存不足！\033[0m"
```

### 五、注意事项
1. **引号的选择**：
    - 单引号 `''`：原样输出，不解析任何变量/转义字符，适合固定字符串；
    - 双引号 `""`：解析变量、转义字符，是最常用的方式；
    - 无引号：易被空格/特殊字符分割，仅适合极简单的字符串。
2. **特殊字符转义**：
    - 若要输出 `$`、`\`、`*` 等特殊字符，需用 `\` 转义（或用单引号）：
      ```bash
      echo "输出\$符号"  # 输出：输出$符号
      echo '输出*符号'    # 输出：输出*符号
      ```
3. **与 printf 的区别**：
    - `echo`：简单易用，适合快速输出；
    - `printf`：格式控制更强（类似 C 语言 printf），适合复杂格式化输出。

### 总结
1. `echo` 核心作用是输出字符串/变量，基础用法：`echo [内容]`，常用选项 `-n`（不换行）、`-e`（解析转义）；
2. 引号使用原则：单引号原样输出，双引号解析变量，推荐优先用双引号；
3. 实战高频场景：重定向写入文件、命令替换输出结果、管道传递内容、彩色文字提示。
