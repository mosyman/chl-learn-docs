
下面给你一份**Linux Vim 超详细、最实用用法**，从入门到常用操作全覆盖，直接照着用就行。

---

# 一、Vim 三种模式（必须懂）
1. **命令模式（Normal）**
    - 刚打开 Vim 就是这个模式
    - 用来移动、删除、复制、粘贴
2. **插入模式（Insert）**
    - 用来**打字编辑**
3. **底行模式（Command-line）**
    - 输入 `:` 进入
    - 用来保存、退出、查找、替换、设置

---

# 二、最基础：打开/保存/退出
## 打开文件
```bash
vim 文件名
vim 1.txt
```

## 底行模式命令（输入 : 后敲）
- `:w`        保存
- `:q`        退出（没修改才能退）
- `:wq`       保存并退出
- `:q!`       强制退出不保存
- `:w!`       强制保存
- `:x`        保存退出（同 wq）

---

# 三、进入编辑模式（打字）
命令模式下按：
- `i`   在光标**前**插入
- `I`   行**首**插入
- `a`   光标**后**插入
- `A`   行**尾**插入
- `o`   下一行新开一行
- `O`   上一行新开一行
- `Esc` 回到命令模式

---

# 四、光标移动（命令模式）
## 基础移动
- `h` 左
- `j` 下
- `k` 上
- `l` 右

## 快速移动
- `0`   行首
- `$`   行尾
- `gg`  文件开头
- `G`   文件末尾
- `:n`  跳到第 n 行（例 :10）
- `Ctrl + f` 下一页
- `Ctrl + b` 上一页

---

# 五、删除/剪切
- `x`      删除光标处字符
- `dd`     删除整行（剪切）
- `dw`     删除一个单词
- `d0`     删除到行首
- `d$`     删除到行尾
- `ndd`    删除 n 行（例 3dd）
- `D`      从光标删到行尾

---

# 六、复制/粘贴
- `yy`     复制当前行
- `nyy`    复制 n 行（3yy）
- `yw`     复制一个单词
- `p`      粘贴到光标**后**
- `P`      粘贴到光标**前**

---

# 七、撤销/恢复
- `u`      撤销
- `Ctrl + r` 恢复撤销

---

# 八、查找
- `/关键词`   向下查找
- `?关键词`   向上查找
- `n`        下一个匹配
- `N`        上一个匹配

---

# 九、替换（底行模式）
## 替换当前行
```
:s/旧/新
```

## 替换当前行所有
```
:s/旧/新/g
```

## 全文替换
```
:%s/旧/新/g
```

## 全文替换并确认
```
:%s/旧/新/gc
```

---

# 十、代码常用操作
- `>>`  向右缩进
- `<<`  向左缩进
- `==`  自动格式化缩进
- `gg=G` 全文格式化

---

# 十一、多文件/分屏
## 分屏
- `:sp 文件名`  水平分屏
- `:vsp 文件名` 垂直分屏
- `Ctrl + w w`  切换窗口

## 多标签
- `:tabnew 1.txt` 新建标签
- `gt` 下一个标签
- `gT` 上一个标签

---

# 十二、显示行号/设置
```
:set nu        显示行号
:set nonu      关闭行号
:set autoindent 自动缩进
:set tabstop=4  tab=4空格
:syntax on     语法高亮
```

永久保存配置：
```bash
vim ~/.vimrc
```
写入：
```
set nu
set autoindent
set tabstop=4
syntax on
```
