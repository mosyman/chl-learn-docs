# 一、最核心文件命令（必须全会）
## 1. ls —— 列出文件/目录
```bash
ls                # 简单列表
ls -l             # 详细信息（权限、大小、时间）
ls -a             # 显示隐藏文件（.开头）
ls -lh            # 大小人性化显示（KB/MB/GB）
ls -lt            # 按修改时间排序（最新在前）
ls -la            # 所有文件+详细信息
ls -d */          # 只看目录
```

## 2. cd —— 切换目录
```bash
cd /home          # 绝对路径
cd ..             # 回到上一级
cd ~              # 回家目录
cd -              # 回到上一次所在目录
cd                # 默认回家
```

## 3. pwd —— 显示当前路径
```bash
pwd
```

## 4. mkdir —— 创建目录
```bash
mkdir test        # 创建单个目录
mkdir -p a/b/c    # 递归创建多级目录
mkdir -m 755 test # 带权限创建
```

## 5. rm —— 删除文件/目录（危险！）
```bash
rm file           # 删除文件
rm -f file        # 强制删除，不提示
rm -r dir         # 递归删除目录
rm -rf dir        # 强制删目录（最常用）
rm -rf /*         # 千万不要试！删系统
```

## 6. cp —— 复制
```bash
cp a b            # 复制文件a到b
cp -r dir1 dir2   # 复制目录
cp -a dir1 dir2   # 保留所有属性（权限、时间）
cp -v file dest   # 显示过程
```

## 7. mv —— 移动 / 重命名
```bash
mv a b            # 重命名 a → b
mv file /tmp      # 移动文件到 /tmp
mv dir/* .        # 把目录里所有东西移到当前
```

## 8. touch —— 创建空文件 / 更新时间
```bash
touch file        # 文件不存在则创建，存在则更新时间
touch -d "2025-01-01" file
```

# 二、查看文件内容命令
## 9. cat —— 查看全部内容
```bash
cat file
cat -n file       # 显示行号
```

## 10. less —— 分页查看（最推荐）
```bash
less file
# 操作：
# 上下箭头 / 空格翻页 / q退出 / /关键词搜索
```

## 11. more —— 老式分页
```bash
more file
```

## 12. head —— 看文件开头
```bash
head file
head -n 20 file   # 看前20行
```

## 13. tail —— 看文件末尾（日志神器）
```bash
tail file
tail -n 20 file   # 最后20行
tail -f file      # 实时刷新（看日志）
tail -F file      # 文件重建也能继续追
```

# 三、查找/搜索命令
## 14. find —— 查找文件（最强）
```bash
find / -name "*.log"       # 根目录搜索 .log 文件
find . -type f -size +100M # 找大于100M的文件
find . -type d             # 只找目录
find . -mtime -1           # 24小时内修改的文件
find . -exec chmod 644 {} \; # 找到后执行命令
```

## 15. grep —— 在文件里搜索内容
```bash
grep "hello" file
grep -i "error" log        # 忽略大小写
grep -n "test" file        # 显示行号
grep -r "key" .            # 递归搜索目录
grep -v "ignore" file      # 反向匹配（排除）
```

# 四、文件权限/属性命令
## 16. chmod —— 修改权限
```bash
chmod 644 file
chmod 755 script.sh
chmod +x script.sh    # 加执行权限
chmod -R 755 dir      # 递归改目录权限
```

## 17. chown —— 修改所有者
```bash
chown user file
chown user:group file
chown -R user:group dir
```

## 18. stat —— 看文件inode详细信息
```bash
stat file
```

## 19. file —— 查看文件类型
```bash
file filename
```

# 五、文件比较/处理
## 20. diff —— 比较文件差异
```bash
diff a b
```

## 21. wc —— 统计行数/单词数/字节数
```bash
wc -l file    # 行数
wc -w file    # 单词数
wc -c file    # 字节数
```

## 22. sort —— 排序
```bash
sort file
sort -n file  # 数字排序
sort -r file  # 倒序
```

## 23. uniq —— 去重
```bash
sort file | uniq
```

# 七、压缩/解压
## 25. tar —— 打包/解压（最常用）
```bash
tar -zcvf archive.tar.gz dir/   # 压缩
tar -zxvf archive.tar.gz        # 解压
tar -jcvf archive.tar.bz2 dir/
tar -jxvf archive.tar.bz2
```

## 26. gzip / bzip2
```bash
gzip file
gunzip file.gz
```
