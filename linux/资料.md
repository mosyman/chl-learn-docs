## 一、Linux 官方权威资料

### 1. Linux 内核官方文档（最底层权威）
- **Linux Kernel Documentation**（内核源码级文档）
    - 地址：https://docs.kernel.org/
    - 内容：内核架构、系统调用、设备驱动、内核参数、管理指南、开发流程等。
    - 适合：深入理解 Linux 底层、系统调优、内核开发。
- **Linux man-pages（命令手册，最常用）**
    - 终端直接查：`man ls`、`man grep`、`man bash`
    - 在线版：https://man7.org/linux/man-pages/
    - 说明：所有命令、系统调用、库函数的权威定义，是日常查用法的第一选择。

### 2. Linux 文档项目（TLDP，经典权威）
- **The Linux Documentation Project (TLDP)**
    - 地址：https://www.tldp.org/
    - 核心经典（免费在线）：
        - **《Linux Installation and Getting Started》**：入门安装与基础使用
        - **《Linux System Administrators Guide》**：系统管理权威指南
        - **《Bash Guide for Beginners》**：Bash 入门权威教程
        - **《Advanced Bash-Scripting Guide》**：Bash 脚本进阶圣经
    - 特点：社区维护、历史悠久、覆盖从入门到内核开发。

### 3. 主流发行版官方文档（最实用）
- **Ubuntu**（最流行桌面/服务器）
    - 官方文档：https://ubuntu.com/server/docs
    - 中文社区：https://wiki.ubuntu.org.cn/
- **CentOS / RHEL**（企业级主流）
    - RHEL 官方：https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/
    - CentOS 文档：https://docs.centos.org/
- **Debian**（稳定发行版标杆）
    - 官方手册：https://www.debian.org/doc/

### 4. Linux 基金会（行业标准与培训）
- **Linux Foundation Training**（免费/付费权威课程）
    - 地址：https://training.linuxfoundation.org/
    - 免费课：**Linux Foundation Certified SysAdmin (LFCS)** 预备材料、开源导论等
- **Filesystem Hierarchy Standard (FHS)**（Linux 文件系统标准）
    - 地址：https://refspecs.linuxfoundation.org/FHS_3.0/fhs/index.html
    - 说明：定义 `/bin`、`/etc`、`/home`、`/var` 等目录规范，理解 Linux 结构的基础。

## 二、Shell（Bash）官方权威资料

### 1. GNU Bash 官方手册（Bash 终极权威）
- **GNU Bash Reference Manual**
    - 地址：https://www.gnu.org/software/bash/manual/bash.html
    - 内容：Bash 语法、内置命令、变量、函数、重定向、作业控制、脚本编写等完整定义。
    - 适合：所有 Bash 学习阶段，查语法、特性、行为的唯一权威。
- **Bash 官方主页与 FAQ**
    - 主页：https://www.gnu.org/software/bash/
    - FAQ：https://tiswww.case.edu/php/chet/bash/FAQ（维护者 Chet Ramey 维护）

### 2. POSIX 标准（Shell 语言国际标准）
- **The Open Group POSIX.1-2024 Shell & Utilities**
    - 地址：https://pubs.opengroup.org/onlinepubs/9799919799.2024edition/utilities/V3_chap02.html
    - 内容：**Shell Command Language** 标准定义（`sh` 语法、变量、扩展、重定向等）。
    - 说明：所有 Unix/Linux Shell 的语法基础，写可移植脚本必看。

### 3. 权威进阶指南
- **Advanced Bash-Scripting Guide (TLDP)**
    - 地址：https://tldp.org/LDP/abs/html/
    - 特点：从基础到高级脚本、文本处理（`sed`/`awk`）、正则、调试、最佳实践，中文社区有翻译。
- **Bash Guide (Greg Wooledge)**
    - 地址：https://mywiki.wooledge.org/BashGuide
    - 特点：强调**正确用法与避坑**，适合进阶与写出健壮脚本。

## 三、学习路径与资料使用建议

### 1. 入门阶段（先掌握命令行与基础）
1. 先看：**man-pages**（日常查命令）+ 发行版官方入门文档（如 Ubuntu 入门）。
2. 再读：**TLDP《Bash Guide for Beginners》** 或 **GNU Bash 手册基础章节**。
3. 配合：**《Linux 命令行大全》**（阮一峰，中文友好）https://wangdoc.com/bash/

### 2. 进阶阶段（系统管理 + 脚本开发）
1. 系统管理：**TLDP《Linux System Administrators Guide》** + 发行版服务器文档。
2. Shell 脚本：**Advanced Bash-Scripting Guide** + **GNU Bash 手册完整阅读**。
3. 标准与规范：**FHS** + **POSIX Shell 标准**。

### 3. 深入阶段（底层与内核）
- **Linux Kernel Documentation**（内核文档）
- **man 2/3**（系统调用与库函数手册）
- **Linux 内核相关书籍**（如《Linux Kernel Development》）
