LangChain4j 是一个**社区驱动的独立开源项目**，并非由单一商业公司全资拥有或控制。

### 一、核心团队与创始人
- **创始人 & 项目负责人**：**Dmytro Liubarskyi**（乌克兰开发者）。
- **核心维护者**：由 Dmytro 带领的独立开源团队，负责项目的整体架构、 roadmap 与核心代码维护。
- **项目定位**：它是 LangChain 理念的 **Java 独立实现**，**并非 LangChain Inc.（美国公司）的官方分支或子项目**。

### 二、主要支持与贡献方（公司/组织）
LangChain4j 得到多家科技巨头的深度赞助与贡献，形成了强大的企业级支持网络：
1. **Red Hat（红帽）**
    - 最主要的企业支持者之一，多位工程师（如 Georgios Andrianakis、Eric Deandrea、Clement Escoffier）是核心贡献者。
    - 深度推动与 `Quarkus`（Red Hat 旗下云原生 Java 框架）的集成。
2. **Microsoft（微软）**
    - 2025 年起成为战略合作伙伴，提供大量代码贡献、安全审计与 Azure 生态深度集成（Azure OpenAI、Azure AI Search、Cosmos DB 等）。
    - 微软 Java 开发者团队贡献了数百个 Issue 与 PR。
3. **其他重要贡献公司**
    - **IBM**：Java Champion 与资深工程师参与开发。
    - **MongoDB**：官方集成 MongoDB Atlas 向量搜索。
    - **AWS、Google Cloud、阿里云**：提供各大云厂商模型与服务的集成模块。

### 三、项目治理与归属
- **代码托管**：GitHub 独立组织 `langchain4j`（https://github.com/langchain4j ）。
- **许可证**：**Apache 2.0** 开源协议，完全开放、可商用。
- **治理模式**：社区驱动，核心决策由维护者团队与贡献者社区共同做出。

---

简单总结：**LangChain4j 由独立创始人 Dmytro Liubarskyi 发起并主导，是一个社区项目；Red Hat、Microsoft 等是其最大的企业级支持者与贡献方，但项目本身不归属于任何单一公司**。
