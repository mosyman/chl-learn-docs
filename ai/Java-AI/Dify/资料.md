Dify 是由**苏州语灵人工智能科技有限公司**开发的开源 LLM 应用开发平台，采用开放核心（Open Core）模式，聚焦 LLMOps 全生命周期，帮助开发者与企业快速构建生产级 AI 应用与智能体（Agent）。

---

### 核心背景速览
| 项目 | 详情 |
| :--- | :--- |
| **公司与创始人** | 苏州语灵（2023-03-23 注册）；CEO 张路宇（前腾讯云 CODING DevOps 产品负责人） |
| **团队基因** | 核心成员来自腾讯云 CODING，深耕开发者工具与企业服务 |

---

### 发展关键节点
1. **2023-03** 团队成立，启动研发。
2. **2023-05** 开源版本发布，首日 240+ 开发者参与共建。
3. **2024-03** 集成百度千帆，推出 Agent 框架。
4. **2024-11** 上线企业套件（知识库版本管理、多租户、审计），进入金融基础设施。
5. **2025-03** 与 NVIDIA 合作，发布 v1.0.0，推出插件市场与多模态工作流。

---

### 产品定位与核心能力
- **定位**：低代码可视化编排 + 生产级 LLMOps，覆盖从原型到规模化运营的全链路。
- **核心功能**：RAG 管道、AI 工作流、Agent 框架、模型管理、可观测性、私有化部署。
- **生态兼容**：支持 OpenAI、Anthropic、百度千帆、AWS Bedrock、NVIDIA 等；主流模型 48 小时内完成接入。

---

### 商业模式
- **社区版**：开源免费，提供基础能力，适合个人与中小团队。
- **企业版**：闭源，包含 SSO、高级权限、合规审计、专属支持等。
- **云服务**：托管式部署，降低运维成本，面向企业客户。

---

### 总结
Dify 由前腾讯云 CODING 核心团队创立，依托开发者工具基因与开源策略快速崛起，获得阿里云等资本支持，在 LLMOps 领域形成“易用性+生产级”的差异化优势，已成为全球主流的 LLM 应用开发平台之一。
