微服务引擎MSE（Microservices Engine）是一个面向业界主流开源微服务生态的一站式微服务平台，
提供注册配置中心（原生支持Nacos/ZooKeeper/Eureka）、
云原生网关（原生支持Ingress/Envoy）、
微服务治理（原生支持Spring Cloud/Dubbo/Sentinel，遵循OpenSergo服务治理规范）的能力。
