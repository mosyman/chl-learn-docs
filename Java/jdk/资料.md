

[oracle.jdk官方文档](https://docs.oracle.com/en/java/javase/17/index.html)




学习 Java 最权威、最核心的官方资源，全部由 **Oracle（Java 官方维护方）** 提供，分为**官方教程**、**API 文档**、**语言/虚拟机规范**三大类，下面按学习阶段整理入口与用法：

---

### 一、官方入门/进阶教程（The Java™ Tutorials）
最适合从零学 Java 的官方教程，结构清晰、带大量可运行示例，覆盖从基础到高级的完整路径。

- **入口**：https://docs.oracle.com/javase/tutorial/
- **核心模块（按学习顺序）**：
    1. **Getting Started**：安装 JDK、写第一个 Java 程序、开发环境配置
    2. **Learning the Java Language**：语法、面向对象、异常、泛型、集合、并发等核心知识
    3. **Essential Java Classes**：IO、日期时间、正则、国际化等常用类库
    4. **Collections**：集合框架（List/Set/Map）完整用法与最佳实践
    5. **Concurrency**：线程、线程池、同步、并发工具类（Java 并发核心）
    6. **JDBC**：数据库连接与操作
    7. **JavaFX/Swing**：桌面 GUI 开发（可选）
- **特点**：全英文、官方示例可直接复制运行、知识点权威无偏差。

---

### 二、Java SE API 文档（Javadoc）
开发时**必查**的标准库手册，包含所有 Java 内置类/接口/方法的详细说明、参数、返回值、异常、版本兼容信息。

#### 1. 常用版本入口（LTS 优先）
- **Java 21（最新 LTS）**：https://docs.oracle.com/en/java/javase/21/docs/api/
- **Java 17（主流 LTS）**：https://docs.oracle.com/en/java/javase/17/docs/api/
- **Java 8（企业常用）**：https://docs.oracle.com/javase/8/docs/api/
- **最新版（JDK 25）**：https://docs.oracle.com/en/java/javase/25/docs/api/

#### 2. 文档结构与用法
- 按**模块（Module）→包（Package）→类（Class）** 层级组织
- 每个类页面包含：继承关系、构造方法、实例/静态方法、字段、异常说明、示例代码
- 支持**搜索**（右上角放大镜），直接查类名/方法名（如 `String`、`ArrayList`）

#### 3. 离线使用
- 下载对应 JDK 版本的 **Java SE Documentation** 压缩包，本地解压后用浏览器打开 `index.html`
- IDE（IDEA/Eclipse）可直接关联本地/在线 Javadoc，编码时一键查看

---

### 三、Java 语言与虚拟机规范（权威定义）
深入理解 Java 底层原理的**终极权威资料**，适合进阶与源码研究。

1. **Java Language Specification（JLS，Java 语言规范）**
    - 定义 Java 语法、语义、类型系统、编译规则
    - 入口：https://docs.oracle.com/javase/specs/jls/se21/html/index.html
2. **Java Virtual Machine Specification（JVMS，Java 虚拟机规范）**
    - 定义 JVM 结构、字节码、类加载、内存模型、执行引擎
    - 入口：https://docs.oracle.com/javase/specs/jvms/se21/html/index.html

---

### 四、JDK 工具与高级文档
JDK 自带工具（`javac`/`java`/`jar`/`jlink`/`jconsole` 等）的官方手册，以及 JVM 调优、监控、安全等高级指南。

- **入口**：https://docs.oracle.com/en/java/javase/21/docs/
- 核心内容：
    - JDK 工具规范（Tool Specifications）
    - JVM 调优、垃圾回收（Garbage Collection Tuning）
    - 监控与管理（JMX、Flight Recorder）
    - 安全编码指南（Secure Coding Guidelines）
    - JShell 交互式编程指南

---

### 五、OpenJDK 官方资源（源码与社区）
想研究 Java 底层实现、看 JDK 源码，用 OpenJDK 官网：
- **OpenJDK 主页**：https://openjdk.org/
- **源码仓库**：https://github.com/openjdk/jdk
- **JEP 提案**：Java 新特性的设计与演进文档（https://openjdk.org/jeps）

---

### 六、学习路径建议（按阶段）
1. **入门**：先看 **The Java Tutorials → Getting Started + Learning the Java Language**，配合 IDE 写代码
2. **开发**：随时查阅 **API 文档**，解决类/方法用法问题
3. **进阶**：读 **JLS/JVMS** 理解底层，看 **OpenJDK 源码** 研究实现
4. **实战**：结合 **JDK 工具文档** 做调试、调优、打包部署
